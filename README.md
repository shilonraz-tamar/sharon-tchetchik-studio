## 🚀 התקנה מהירה (3 דקות, פעם אחת בלבד)

### שלב 1: התקנת Claude Code
פתחי **Terminal** במק (Cmd+Space → "Terminal" → Enter), והדביקי:

```bash
curl -fsSL https://claude.ai/install.sh | sh
```

### שלב 2: התקנת הפלאגין שלך
הדביקי את שתי השורות הבאות (פקודה אחת בעצם):

```bash
claude plugin marketplace add shilonraz-tamar/claude-code-sharon-studio
claude plugin install sharon-studio@claude-code-sharon-studio
```

### שלב 3: להתחיל לדבר
בטרמינל, הדביקי:
```bash
claude
```

ושאלי מה שתרצי 💕
